# AtChat
基于XMPP聊天APP-IOS(OC)项目

## 安装说明
1. pod isntall
2. 服务器端安装openfire
3. AtChat.pch文件中IP地址改为服务器地址

## 功能
* [x] 登陆注册(注册以手机号码注册、短信验证)
* [x] 发送消息
* [x] 接收消息
* [x] 接收好友请求
* [x] 发起好友请求
* [x] 图片消息
* [x] 语音消息
